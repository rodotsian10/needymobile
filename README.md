## 🇬🇧 English

A Web-based Desktop OS replica inspired by the visual novel, **[NEEDY STREAMER OVERLOAD](https://store.steampowered.com/app/1451940/NEEDY_STREAMER_OVERLOAD/)** (NEEDY GIRL OVERDOSE).

This project is a React-based web port heavily inspired by the popular [Needy Streamer Overload Rainmeter Skins](https://github.com/lezzthanthree/Needy-Streamer-Overload) by lezzthanthree, bringing the authentic in-game desktop experience directly into your web browser.

### ✨ Features
- **Authentic Boot Sequence**: Classic DOSE OS boot screen with startup sounds and continuous BGM.
- **Ame-chan Webcam**: Fully animated Ame sprite idling in her room.
- **KAngel Transformation**: Click the "on Air" button to watch the dynamic zoom-in transformation sequence into KAngel.
- **JINE Messaging App**: Fully functional chat interface mimicking the JINE app, complete with sending messages and automated replies from Ame.
- **Task Manager**: A real-time status window that calculates the current "DAY" based on the real-world date, displaying Stress, Affection, and Darkness bars.
- **Draggable Windows**: All apps (Webcam, JINE, Task Manager) can be freely dragged and positioned around the retro desktop.

### 🚀 Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` in your browser.

<a name="한국어"></a>
## 🇰🇷 한국어

비주얼 노벨 **[NEEDY STREAMER OVERLOAD (니디 걸 오버도즈)](https://store.steampowered.com/app/1451940/NEEDY_STREAMER_OVERLOAD/)**에서 영감을 받아 제작된 웹 기반 데스크톱 OS 레플리카입니다.

이 프로젝트는 lezzthanthree님이 제작하신 유명한 [니디 걸 오버도즈 레인미터 스킨](https://github.com/lezzthanthree/Needy-Streamer-Overload)을 React로 포팅한 웹 이식판으로, 원작 게임 특유의 데스크톱 환경을 웹 브라우저 안에서 완벽하게 재현합니다.

### ✨ 주요 기능
- **고증 부팅 시퀀스**: 시작음과 연속적인 BGM이 포함된 클래식 DOSE OS 부팅 화면.
- **아메쨩 웹캠**: 방 안에서 대기하는 아메쨩의 풀 애니메이션 스프라이트.
- **초텐쨩 변신 모드**: "on Air" 버튼을 클릭하면, 줌인 연출과 함께 초텐쨩으로 변신하는 역동적인 애니메이션 재생.
- **JINE 메신저**: 원작의 JINE 앱을 구현하여, 메시지를 전송하고 아메의 자동 답장을 받을 수 있는 기능.
- **작업 관리자**: 현실 날짜를 기반으로 현재 "DAY(일차)"를 계산하며, 스트레스, 호감도, 우울도 바를 보여주는 실시간 상태창.
- **드래그 가능한 윈도우 창**: 모든 앱(웹캠, JINE, 작업 관리자)을 레트로 감성의 바탕화면 위에서 자유롭게 드래그하고 배치 가능.

### 🚀 로컬 실행 방법

1. 패키지 의존성 설치:
   ```bash
   npm install
   ```
2. 개발 서버 실행:
   ```bash
   npm run dev
   ```
3. 브라우저에서 `http://localhost:5173` 접속.
