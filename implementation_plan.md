# DOSE OS 대규모 업데이트 기획 및 구현 계획서

사용자님이 구상하신 훌륭한 아이디어들을 바탕으로 향후 업데이트될 기능들을 체계적으로 정리했습니다. 각 기능의 현재 상태와 구현 난이도, 구체적인 작업 방향을 확인해 주시기 바랍니다.

## 1. 현재 구현 상태 요약

### ✅ 현재 구현된 기능 (완료)

- **기본 OS 인터페이스**: 부팅 화면, 데스크탑, 아이콘, 바탕화면, 드래그 가능한 윈도우 창 시스템
- **초텐짱/아메쨩 웹캠**: 큐비즘 애니메이션 렌더링, 수동 모션 전환, 상태 변화 (Idle <-> KAngel)
- **JINE (채팅 앱)**: 기본적인 UI, 단순 로컬 채팅 더미 데이터 구조
- **작업 관리자 (Status)**: 하드코딩된 상태 게이지 표시 (스트레스, 호감도, 다크니스 등)
- **모션 디렉터**: 수동으로 모션을 클릭해서 변경하는 관리자 앱
- **모바일/반응형 최적화**: 터치 드래그 지원 및 줌 방지

### ❌ 아직 구현되지 않은 기능 (이번 계획 대상)

- **설정 앱**: 사운드, JINE 연동, 디렉터 설정 등을 제어하는 중앙 관리 시스템 부재
- **전역 상태 동기화 (모션 자동화)**: 앱의 동작(음악 재생, 메모 작성)에 따라 펫의 모션이 자동으로 바뀌는 유기적인 시스템 부재
- **뮤직 플레이어**: 오디오 재생 기능 및 유튜브 API 연동 부재
- **메모장**: 로컬 스토리지(브라우저 저장소)를 활용한 텍스트 저장 기능 부재
- **기능형 작업 관리자**: 실제 PC의 볼륨 등과 연동되는 동적 게이지 부재

## 2. 기능별 상세 구현 계획 및 난이도

### 1️⃣ 설정(Settings) 앱 추가

**구현 난이도: ⭐⭐⭐ (보통)**

- **목표**: OS 전반의 동작을 제어하는 중앙 제어판 앱 생성
- **상세 내용**:
  - **사운드 설정**: 전역 BGM 및 효과음 재생 객체를 만들고 On/Off 토글 (기본: On)
  - **JINE 설정**: OpenAI API Key 입력 폼, 아메/초텐짱 선톡(자동 말걸기) On/Off 스위치 (기본: Off)
  - **디렉터 설정**:
    - 바탕화면의 '모션 디렉터' 아이콘 숨기기/보이기 기능 (기본: Off - 숨김)
    - **모션 자동변경**: 뮤직 플레이어나 메모장 사용 시 펫의 모션을 상황에 맞게 덮어씌우는 로직 (기본: On)
  - **기본 설정 (보류)**: 알림 설정, 화면보호기 설정 UI만 배치해 두고 추후 구현

### 2️⃣ 작업 관리자(Task Manager) 패치

**구현 난이도: ⭐⭐ (쉬움)**

- **목표**: 기존 더미 데이터로 있던 'Stress' 항목을 실제 의미 있는 데이터로 변경
- **상세 내용**:
  - 기존 'Stress' 게이지 라벨을 'Volume (볼륨)'으로 변경
  - 설정 앱이나 뮤직 플레이어의 사운드 설정과 연동하여 현재 볼륨 퍼센티지(%)를 시각적으로 표시

진행 끝!

### 3️⃣ 메모장(Notepad) 앱 추가

**구현 난이도: ⭐⭐⭐ (보통)**

- **목표**: 텍스트 작성 및 저장이 가능한 메모장 구현
- **상세 내용**:
  - `textarea`를 활용한 윈도우 95 스타일의 메모장 창 UI
  - 브라우저의 `localStorage`를 활용하여 새로고침해도 메모가 유지되도록 저장 기능 구현
  - 메모장 활성화/타이핑 시 초텐짱 모션을 '타이핑' 모션 등으로 자동 전환 (설정 연동)
  - _(모바일 위젯 기능은 현재 OS 구조상 보류)_

### 4️⃣ 뮤직 플레이어(Music Player) 추가

**구현 난이도: ⭐⭐⭐⭐⭐ (매우 어려움)**

- **목표**: 인게임 브금 및 유튜브 영상을 재생할 수 있는 전용 플레이어 앱
- **상세 내용**:
  - **로컬 재생**: '큐비즘', '큐비비비즘', '오버도즈', '야메로' 등 내장 오디오 파일 재생 제어 (재생/일시정지/볼륨/트랙 이동)
  - **유튜브 연동**: `react-youtube` 패키지 또는 iframe API를 활용해 온라인 음악 스트리밍 지원. (※ 유튜브 정책상 백그라운드 자동 재생 등 제약이 많아 꼼꼼한 처리가 필요함)
  - **모션 자동 전환**: 노래가 재생되는 동안 초텐짱의 모션을 '헤드폰(음악 감상)' 또는 '광기(리듬 타기)' 모션 등으로 자동 변경

## 3. 개발 진행 순서 (추천 로드맵)

기능들이 서로 얽혀 있기 때문에(예: 사운드 설정 ↔ 뮤직 플레이어, 자동 모션 설정 ↔ 모든 앱), 다음과 같은 순서로 기반을 다지며 개발하는 것을 추천합니다.

1. **Phase 1: 상태 관리 확장 및 작업 관리자 수정**
   - 글로벌 스토어(`useAppStore`)에 설정값(사운드, 볼륨, 디렉터 모드 등)을 저장할 변수 추가
   - 작업 관리자 UI를 볼륨 연동으로 패치
2. **Phase 2: 설정 앱 UI 및 연동**
   - '설정' 아이콘과 윈도우 창 제작
   - JINE API 키 입력, 사운드 On/Off, 모션 디렉터 아이콘 숨기기 기능 연동 완료
3. **Phase 3: 메모장 앱**
   - 메모장 UI 및 `localStorage` 저장 기능 개발
   - '모션 자동변경' 설정과 연동하여 메모 시 모션 전환 테스트
4. **Phase 4: 뮤직 플레이어 (최종 보스)**
   - 플레이어 UI 제작 및 로컬 음악 재생 구현
   - 유튜브 연동 모듈 추가
   - 재생 상태에 따른 모션 연동 완결
